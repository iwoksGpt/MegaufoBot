# فایل‌های تبدیل‌شده

۷ فایل ارسالی کاربر که با فرمت `.txt` بودند، برای پروژه‌ی Python Telegram Bot به `.py` تبدیل و دسته‌بندی شدند:

- `config.txt` → `megaufobot/config.py`
- `main.txt` → `megaufobot/main.py`
- `database.txt` → `megaufobot/database.py`
- `tmdb_api.txt` → `megaufobot/tmdb_api.py`
- `keyboards.txt` → `megaufobot/keyboards.py`
- `messages.txt` → `megaufobot/messages.py`
- `admin_panel.txt` → `megaufobot/admin_panel.py`

برای رعایت امنیت، کلید TMDB و توکن تلگرام داخل کد هاردکد نشده‌اند و باید در `.env` قرار بگیرند.
